# CCPA MSW Coursework Pressure Map

Interactive static visualization for mapping MSW coursework, field placement, and optional outside-effort workload across a full-time two-year program.

## Open Locally

Open `index.html` in a browser, or serve the folder with a simple local web server:

```sh
python3 -m http.server 4173
```

Then visit `http://127.0.0.1:4173/`.

## Current Scope

- Year 1 Fall and Spring foundation coursework
- Year 2 Fall and Spring advanced coursework
- Field instruction workload assumptions
- Optional student reality overlay for paid work and outside commitments
- Filters by term, course, assignment type, group work, writing-heavy work, graded work, and major assignments
- Restructure mode for moving assignment cards within course rows
